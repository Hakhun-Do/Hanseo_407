## 🏫 1. 한서대학교 FANTA 연구실 (2024)

2024년 한서대학교 FANTA 연구실에서 진행한 학술 활동, 세미나 및 연구 프로젝트 소스 코드와 요약 문서들을 정리한 공간입니다.

### 📅 주요 활동 및 연구 내용
* **연구실 프로젝트**: (소방와방 프로젝트, DB CAR RENT 프로젝트, 홈트 앱 프로젝트, WHERE 앱 프로젝트)
* **기술 세미나**: 연구실 내에서 진행한 기술 공유 및 발표 자료 관리
* **개인 학습 아카이브**: 데이터베이스, 자료구조 등 직무 역량 강화를 위한 연구 데이터 관리

## ⚡ 2. 백준 알고리즘 문제풀이 (BOJ)
매일 꾸준하게 백준(Baekjoon Online Judge) 알고리즘 문제를 풀이하고 코드를 기록하는 공간입니다. 
문제의 핵심 아이디어와 풀이 과정을 주석으로 명시하여 코드의 가독성을 높이고 있습니다.

### 🛠️ 주로 사용하는 언어 & 도구
* **Language**: Java / C++
* **IDE**: IntelliJ / VS Code
